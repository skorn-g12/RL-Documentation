Orion.AI

The final goal is to be able to come up with a myriad of Minecraft bots that do specialized tasks. But let’s start at the beginning. We will be using OpenAI’s gym for our experimentation. Even MineRL’s APIs are just like other gym environments.

Note

All development of Gym has been moved to Gymnasium, a new package in the Farama Foundation that’s maintained by the same team of developers who have maintained Gym for the past 18 months. So if you see Gym anywhere in the doc, it just means Gymnasium : )
